# Privacy Policy

**Effective Date: April 22, 2026**

This Privacy Policy describes how the **Cash Register** application ("the Application") handles your information.

## 1. Information Collection
The Application is a local desktop ledger tool. It **does not collect, transmit, or store any personal data** on any external servers. 

All data entered into the Application (transactions, balances, and dates) is stored locally on your device in a file named `cash_register_data.json` located in your user home directory.

## 2. Information Usage
Since the Application operates entirely offline, your data is never shared, sold, or used for any purpose other than providing the core functionality of the ledger.

## 3. Third-Party Services
The Application does not integrate with any third-party services, analytics, or advertising platforms.

## 4. Data Security
Your data is as secure as your device. We recommend following standard security practices for your operating system to protect your local files.

## 5. Changes to This Policy
We may update this Privacy Policy from time to time. Any changes will be reflected in this document within the project repository.

## 6. Contact Us
If you have any questions about this Privacy Policy, please open an issue in the GitHub repository.
